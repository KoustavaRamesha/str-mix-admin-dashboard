# **App Name**: SolidSite Digital

## Core Features:

- Statically Generated Pages: Deliver lightning-fast Home, About, Services, and Project pages, built at compile time for maximum performance and SEO.
- Incremental Static Blog: Showcase blog listings and individual articles using Incremental Static Regeneration (ISR), allowing new posts to go live instantly without a full site rebuild.
- Client-side Review Submission: Enable users to submit reviews through a client-rendered form, secured with reCAPTCHA v3 for spam prevention.
- Secure Contact Form: Provide a client-side contact form with robust validation and reCAPTCHA protection to queue messages securely for support.
- Admin Dashboard Interface: A secure, client-rendered admin dashboard to manage blog content, moderate reviews, and track support tickets via interaction with the PostgreSQL backend.
- AI Content Creation Tool: An AI-powered tool within the admin dashboard to assist in generating draft blog post content or synthesizing summaries from user reviews to inform content strategy.
- User Authentication & Access: Client-side management of user sessions, login/logout, and routing logic to protect administrative areas using JWTs.

## Style Guidelines:

- The visual scheme adopts a bold, industrial aesthetic using yellow, black, and gray, reflecting strength and modern efficiency.
- Primary color: A vibrant, impactful yellow (#FFD700) to highlight key information, call-to-actions, and interactive elements, conveying energy and focus.
- Background color: A deep, rich black (#1A1A1A) providing strong contrast and a sophisticated foundation for the other colors.
- Accent color: A versatile medium gray (#808080) used for secondary text, borders, and subtle UI elements, offering balance and sophistication.
- Headlines will utilize 'Space Grotesk' (sans-serif) for a modern, slightly tech-inspired, industrial feel that commands attention.
- Body text and general interface elements will use 'Inter' (sans-serif), offering excellent legibility and a clean, objective aesthetic consistent with a robust digital platform.
- Icons will follow a minimalist, geometric style, primarily in outline form, to convey clarity and a modern industrial precision.
- The layout is modular and responsive, ensuring a consistent and sturdy presentation across all devices. Content areas will feature generous spacing for readability.
- Subtle, smooth transition animations will be used for state changes and content loading, enhancing the user experience without being distracting or overtly playful.
